Citymapper

Citymapper is a travelling application where the system will calculate and provide the user a shorten travel route to a destination and optional transit with real-time disruption alerts for each transportation. The objects on it application have a good arrangement, but some contents are using thin text or colour contrast ratio that does not meet the accessibility standard. This will be an issue for users with low vision condition or deficiency, they may find it uncomfortable and difficult to read.

Furthermore, some icons without label or description may confuse the user of their first time, but the application was using a general icon to represent it button.

However, iOS provide accessibility feature for boost user accessibility over the application. Zoom is another function that users can use for increasing their readability and visibility. Users can use this function to zoom on any text that they find difficult to read. In addition, a colour filter also available to use for colour blindness user.

The VoiceOver, built-in screen reader application on iOS was used for testing, which the Citymapper app was supported screen reader and understandable of the word that read aloud or not. The application was compatible with VoiceOver, but it is saying just “button” without any description of the button that is not understandable. This is a critical problem for low vision deficiency and blindness users, they may be unable to use the application efficiently.

According to the Equality Act, the application missing provides many information or feature which requires on the standard. The app does not have section of providing information about application.

The evaluation and test were take place on iOS version 10.3.3 (14G60) in iPhone 6s model MKQN2TH/A.
